# MundialBet Club 2026 ⚽🏆

Una aplicación web para participar en una liga de pronósticos del Mundial 2026. Crea predicciones anónimas, compite con tus amigos y gana premios.

## 🎯 Características

- **Liga de pronósticos** - Participa en una liga privada con tus amigos
- **Anonimato temporal** - Los nombres se revelan cuando el partido inicia
- **Múltiples fases** - Grupos, Octavos, Cuartos, Semifinales y Final
- **Tabla de posiciones** - Ve el ranking en tiempo real
- **Mascotas FIFA 2026** - Zayu (México), Maple (Canadá), Clutch (EE.UU.)
- **PWA** - Instálable como app en tu celular

## 🚀 Cómo usar

### En tu navegador
Abre `index.html` en tu navegador web.

### En tu celular (PWA)

**iPhone:**
1. Abre en Safari
2. Botón compartir (⬆️) → "Añadir a pantalla de inicio"
3. ¡Instalado!

**Android:**
1. Abre en Chrome
2. Menú (⋮) → "Instalar aplicación"
3. ¡Instalado!
